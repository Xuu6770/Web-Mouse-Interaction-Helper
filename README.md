<h1 align = "center">网页交互助手</h1>

## 💡 功能介绍

本脚本意在简化某些操作步骤以小幅提升使用效率和体验，具体功能如下：

### 1. 鼠标左键（拖拽链接/图片在新标签页打开）

针对鼠标左键，主要是为了简化超链接或图片在新标签页打开的操作。默认情况下，在新标签页打开需要右键菜单选择或配合键盘修饰键。

安装本脚本后则可以直接通过**拖拽链接或图片移动任意距离**使其在新标签页打开。

* 脚本没有设置拖拽阈值，拖动即可触发。
* **排除情况**：如果将链接或图片拖入到了可以接收它的对象当中（如搜索框、表单文本框等），则不会触发打开行为。
* **已知局限**：若将图片/链接拖拽至浏览器以外的其它应用中（例如拖到本地资源管理器以保存图片），尽管保存成功，但脚本无法阻止浏览器在新标签页中打开它。

### 2. 鼠标右键（快速复制文本）

针对鼠标右键，主要用于快速复制元素中的文本。

* 点击任何非图片、非表单输入的常规页面元素时，脚本会自动复制该元素内的文本内容。
* **非侵入式设计**：本功能不会阻止鼠标右键本身的默认行为（即弹出网页右键菜单）。

### 3. 鼠标悬停与图片交互

当鼠标指针移动到网页中的图片（宽度 $\ge$ 80px，高度 $\ge$ 50px）上时，图片右上角将自动显示一个半透明且支持磨砂玻璃效果的悬停操作面板，包含以下三个快捷按钮：

* **在新标签页中打开图片**：一键在新标签页中查看大图。
* **复制图片地址**：一键将图片的链接地址复制到剪贴板。
* **下载图片**：直接下载图片到本地。优先使用油猴专用 `GM_download` API 绕过跨域限制，并提供常规 fetch 回退方案，同时支持 Base64 Data URL 下载。

核心优化细节：

1. **参数自动剥离**：在执行上述三项图片操作时，脚本默认会自动清除 URL 中附带的查询参数（如 `?f=webp` 等），从而还原并下载最原本格式的图片（如 `.jpg` / `.png`）。可在脚本代码头部修改 `STRIP_IMAGE_URL_PARAMS = false` 来关闭此功能。
2. **防穿透悬停检测**：通过 CSS 覆写，解决部分网站在图片外层覆盖 `pointer-events: none` 缩放图层而导致无法触发悬停面板的问题。
3. **延迟隐藏防抖**：悬停面板设置了 300ms 延迟隐藏机制，防止用户在将鼠标从图片移往面板时因滑出边缘而导致面板消失。

## ⚙️ 安装使用

1. 为浏览器安装 Tampermonkey 扩展。
2. 点击仓库内的 .js 脚本文件，随后点击右上角的 `Raw` 按钮进行安装。
3. 使用时，可在 Tampermonkey 的脚本菜单中选择性开启或关闭各功能模块。

1. 本项目完全依赖 AI 编写，不保证完美。
2. 本项目仅仅是基于个人习惯和需求实现一些简单的功能，如果出现重复造轮子或雷同，纯属巧合。上传 GitHub 仅供学习交流和分享，并方便自己多端多浏览器同步。
